Docker desktop

起動の仕方

ターミナル（Power Shell）→docker start

ショートカットキー

docker exec -it 名前/bin/bash :Linuxが起動する

su - 名前:ユーザ切り替え

ctrl + L:上に移動し、文字が増える

Tab :文字の補完

ls :ファイルの場所を表示

Tree :全体表示

touch :ファイル作り

mv 名前(今いる場所から) 移動したい場所/:ファイル移動 名前変更

rm :ファイル削除

mkdir :ディレクトリー作成

rmdir :ディレクトリー削除

./* :全てを動かす

↑ :履歴探し

touch 移動したい場所/名前:場所を変えてファイル作成

ll :タイムスタンプを確認

![数字] :履歴にある番号のコマンド実行

cp :ファイル（ディレクトリー）をコピーする

cd :ディレクトリーの場所を移動する

-p :オプションでディレクトリーの下にディレクトリーを作る

-r :再帰的 自分自身を呼び出す（もう一度、終わるまで繰り返す）

ctrl + b :エクスプローラーの表示・非表示

ｄ:ディレクトリー

ｒ:読む

ｗ:書く

ｘ:実行
