JavaScript

変数：データ（値）の入れ物（箱）。箱についている名前が変数名、箱の中に実際の値（文字列、数値）が入っている

let 変数名＝値

=：右辺を左辺に代入する

let ：これから変数を定義します

変数を使うと変数の部分は代入した値に置き換わる

なぜ変数を使うのか：１，同じ値は繰り返し使える２，変更に対応しやすい３，値の意味が分かり易い

命名ルール：1number：数字開始,bango：ローマ字,番号：日本語；不可

number：英単語,oddNumber：2単語以上の場合は大文字で区切る：OK

更新する場合：letが必要なくなり、変数名＝新しい値で良くなる

プログラムは上から順に実行されるため後で代入された値で変数の中身が変わる

  • x = x + 10:x +=10
  • x = x - 10:x -=10
  • x = x * 10:x *=10
  • x = x / 10:x /=10
  • x = x % 10:x %=10

変数は一度代入した値を更新できるが定数は値を更新出来ない

const = 定数
